// tag::purpose[]
:manpurpose: Move the focused window to a named zone (left, center, right)
// end::purpose[]

// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace move-node-to-zone [-h|--help] (left|center|right)

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Zones are active only when the workspace is displayed on an ultrawide monitor (aspect ratio > 2.1:1). If zones are not active, the command is a no-op and prints an error to stderr.

// =========================================================== Options
include::./util/conditional-options-header.adoc[]

-h, --help:: Print help

left|center|right:: The name of the zone to move the focused window into.

// end::body[]
