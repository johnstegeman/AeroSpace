// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace move-node-to-zone [-h|--help] (<zone-id>)

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Zones are active only when the workspace is displayed on an ultrawide monitor (aspect ratio > 2.1:1). If zones are not active, the command is a no-op and prints an error to stderr.

The command updates persistent zone memory for the window's app bundle ID. That remembered assignment is still treated as a fallback: if `[zones.app-routing]` also defines the app, the explicit app route takes precedence on future placements.

// =========================================================== Options
include::./util/conditional-options-header.adoc[]

-h, --help:: Print help

<zone-id>:: The name of the zone to move the focused window into.
